Initial public release of all @midim OSS packages.

- `@midim/shared` — shared MIDI parsing, hashing, and type utilities
- `@midim/fingerprint` — 8-hash deterministic MIDI fingerprinting
- `@midim/diff` — 9-score MIDI similarity with variability level and risk label
- `@midim/poim` — local self-signed POIM certificates (Ed25519 + BLAKE3)
- `@midim/cli` — unified `midim` CLI wrapping all of the above
